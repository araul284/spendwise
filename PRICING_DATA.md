## Notes on Pricing Usage in Audit Engine

1. Annual vs monthly pricing:

   * All calculations normalized to monthly cost.

2. Enterprise plans:

   * Treated as “custom” and excluded from downgrade logic unless strong assumptions can be made.

3. API pricing:

   * Used only for comparative reasoning (e.g., high usage vs low usage).
   * Exact savings require estimated token usage (not assumed unless user provides).

4. Rounding:

   * Savings rounded to nearest whole dollar for clarity.

5. Source of truth:

   * Only official vendor pricing pages used.
   * No third-party aggregators.
